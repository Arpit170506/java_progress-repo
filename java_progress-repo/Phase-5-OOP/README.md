# 🧱 Phase 5: Object-Oriented Programming (OOP)

This phase introduces core Object-Oriented Programming concepts in Java, helping transition from basic coding to structured software development.

---

## 📚 Concepts Covered

* Classes and Objects
* Constructors
* Encapsulation (private variables + getters/setters)
* Inheritance
* Method Overloading
* Method Overriding
* Basic abstraction (intro level)

---

## 💻 Programs Included

1. **Student.java**
   Class representing a student with attributes and display method.

2. **BankAccount.java**
   Simulates a simple bank account with deposit and withdraw functionality.

3. **Car.java**
   Demonstrates object behavior with attributes and methods.

4. **ConstructorExample.java**
   Shows how constructors initialize objects.

5. **EncapsulationExample.java**
   Uses private variables with getters and setters.

6. **InheritanceExample.java**
   Demonstrates inheritance (e.g., Animal → Dog).

7. **MethodOverloading.java**
   Same method name with different parameters.

8. **MethodOverriding.java**
   Child class overriding parent method behavior.

---

## 🎯 Learning Goals

* Understand how real-world entities are modeled in code
* Learn how to organize code using classes and objects
* Apply OOP principles to write scalable programs
* Build a foundation for advanced Java and backend development

---

## 🧠 Notes

* Focus on understanding concepts, not memorizing syntax
* Write small, clear classes before building bigger systems
* OOP is critical for real-world projects and interviews
