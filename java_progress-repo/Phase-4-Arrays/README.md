# 📦 Phase 4: Arrays

This phase focuses on working with arrays to store and manipulate collections of data.

---

## 📚 Concepts Covered

* Array declaration and initialization
* Indexing and traversal
* Iterating using loops
* Basic array operations
* Introduction to problem-solving with collections

---

## 💻 Programs Included

1. **ArraySum.java**
   Calculates the sum and average of array elements.

2. **FindLargest.java**
   Finds the largest element in an array.

3. **SecondLargest.java**
   Finds the second largest element in an array.

4. **EvenOddCount.java**
   Counts even and odd numbers in an array.

5. **ReverseArray.java**
   Reverses the elements of an array.

---

## 🎯 Learning Goals

* Understand how data is stored and accessed in arrays
* Practice traversing arrays using loops
* Build logic for common array-based problems
* Prepare for Data Structures & Algorithms (DSA)

---

## 🧠 Notes

* Arrays are the foundation of most DSA problems
* Focus on writing clean loops and avoiding off-by-one errors
* Try modifying problems (e.g., smallest instead of largest)
