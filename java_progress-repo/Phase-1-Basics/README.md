# 🟢 Phase 1: Java Basics

This phase focuses on building a strong foundation in Java by covering the most essential programming concepts.

---

## 📚 Concepts Covered

* User Input using Scanner
* Output formatting
* Conditional Statements (`if`, `else`, `switch`)
* Loops (`for`, `while`)
* Basic problem-solving logic

---

## 💻 Programs Included

1. **UserInput.java**
   Takes user input (name & age) and prints a formatted message.

2. **EvenOdd.java**
   Checks whether a number is even or odd.

3. **Calculator.java**
   Performs basic arithmetic operations using `switch-case`.

4. **LargestOfThree.java**
   Finds the largest among three numbers.

5. **MultiplicationTable.java**
   Prints the multiplication table of a given number.

6. **SumOfNumbers.java**
   Calculates the sum of first N natural numbers.

7. **Factorial.java**
   Computes factorial of a number using loops.

---

## 🎯 Learning Goals

* Understand how Java programs are structured
* Build confidence with syntax and basic logic
* Learn how to take input and produce output
* Practice writing clean and readable code

---

## 🧠 Notes

* All programs are written using basic constructs only (no advanced concepts)
* Focus is on clarity and understanding rather than optimization
* Each program is written and tested individually
