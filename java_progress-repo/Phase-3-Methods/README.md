# 🧩 Phase 3: Methods (Functions)

This phase focuses on writing modular and reusable code using methods (functions).

---

## 📚 Concepts Covered

* Method declaration and definition
* Parameters and arguments
* Return types
* Void vs non-void methods
* Code reusability and modular design

---

## 💻 Programs Included

1. **CalculatorUsingMethods.java**
   Performs arithmetic operations using separate methods for each operation.

2. **FactorialUsingMethod.java**
   Calculates factorial of a number using a method.

3. **PrimeCheckMethod.java**
   Checks if a number is prime using a boolean-returning method.

4. **TemperatureConverter.java**
   Converts temperature between Celsius and Fahrenheit.

5. **SumUsingMethod.java**
   Returns the sum of two or more numbers using methods.

---

## 🎯 Learning Goals

* Break large problems into smaller reusable methods
* Understand how data flows through parameters and return values
* Improve code readability and maintainability
* Reduce code duplication

---

## 🧠 Notes

* Avoid writing everything inside `main()` — use methods properly
* Use meaningful method names (e.g., `calculateFactorial`, `isPrime`)
* Practice both returning values and printing results
