# 🔁 Phase 2: Loop Mastery & Logic Building

This phase focuses on strengthening problem-solving skills using loops and logical thinking.

---

## 📚 Concepts Covered

* Loop constructs (`for`, `while`)
* Nested loops
* Control flow inside loops
* Number-based logic building
* Pattern generation

---

## 💻 Programs Included

1. **PrimeNumber.java**
   Checks whether a number is prime.

2. **FibonacciSeries.java**
   Generates Fibonacci sequence up to N terms.

3. **ReverseNumber.java**
   Reverses a given number (e.g., 1234 → 4321).

4. **PalindromeNumber.java**
   Checks whether a number is a palindrome.

5. **Pattern1.java**
   Basic star pattern:

   ```
   *
   **
   ***
   ****
   ```

6. **Pattern2.java**
   Number pattern:

   ```
   1
   12
   123
   ```

7. **Pattern3.java**
   Pyramid pattern:

   ```
     *
    ***
   *****
   ```

---

## 🎯 Learning Goals

* Master loop control and execution flow
* Improve logical thinking through pattern problems
* Understand how nested loops work
* Build confidence in solving basic algorithmic problems
